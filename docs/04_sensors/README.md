# アナログセンサーを使う

この章では、ADC0834 を使ってさまざまなアナログセンサーを読み取ります。

## 例

- 可変抵抗: 角度や入力つまみ
- CdS セル: 明るさ
- サーミスタ: 温度
- 土壌水分センサー: 水分量
- 圧力センサー: 押す力

## センサー値の考え方

センサーから得られる値は、最初は単なる ADC の数値です。ADC0834 は 8bit なので、読み取り値は 0 から 255 です。

次の順で考えると理解しやすくなります。

1. ADC0834 の生の値
2. 電圧
3. センサーの物理量
4. プログラムで使う判断値

## キャリブレーション

センサー値は部品差や環境で変わります。最小値と最大値を実測し、プログラム内で扱いやすい範囲に変換します。

```bash
python examples/sensors/calibrate_sensor.py
```
