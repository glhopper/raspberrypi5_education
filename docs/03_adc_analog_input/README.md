# ADC でアナログ値を読む

Raspberry Pi 5 にはアナログ入力がありません。アナログセンサーを使うには ADC を接続します。

## ADS1115

ADS1115 は I2C 接続の 16bit ADC です。精度が高く、低速なセンサー値の読み取りに向いています。

代表的な配線:

| ADS1115 | Raspberry Pi 5 |
| --- | --- |
| VDD | 3.3V |
| GND | GND |
| SCL | GPIO3 / SCL |
| SDA | GPIO2 / SDA |
| A0 | センサー出力 |

## MCP3008

MCP3008 は SPI 接続の 10bit ADC です。8チャンネル入力があり、複数のセンサーを扱いやすい部品です。

## 実験: 可変抵抗を読む

可変抵抗の両端を 3.3V と GND に接続し、中央端子を ADC の入力に接続します。

つまみを回すと、ADC の値と電圧が変化します。
