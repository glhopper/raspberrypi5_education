# 電子回路の基礎

この章では、Raspberry Pi に部品を接続する前に必要な最低限の電子回路を学びます。

## 電圧

電圧は電気を押し出す力のようなものです。Raspberry Pi の GPIO は 3.3V で動作します。

重要なのは、GPIO に 5V を直接入れてはいけないという点です。センサーやモジュールが 5V 出力の場合は、レベル変換や抵抗分圧が必要になります。

## 電流

電流は流れる電気の量です。LED やモーターは電流を必要とします。

GPIO ピンから大きな電流を取り出すことはできません。特にモーターは起動時に大きな電流が流れるため、モータードライバと外部電源を使います。

## 抵抗

抵抗は電流の流れを制限します。LED には電流制限抵抗が必要です。

オームの法則:

```text
V = I * R
```

例: 3.3V で LED に 10mA 流したい場合、LED の順方向電圧を 2.0V とすると、抵抗にかかる電圧は 1.3V です。

```text
R = 1.3V / 0.01A = 130 ohm
```

実際には 150 ohm や 220 ohm など、手元にある近い値を使います。

## GND

GND は回路の基準点です。Raspberry Pi、ADC、センサー、モータードライバ、外部電源を組み合わせるときは、必要に応じて GND を共通にします。

## 抵抗分圧

抵抗分圧は、2本の抵抗で電圧を下げる方法です。5V の信号を 3.3V 以下に下げるときなどに使います。

```text
Vout = Vin * R2 / (R1 + R2)
```

ただし、モーターや大きな電流が流れる用途には使いません。抵抗分圧は信号用です。
