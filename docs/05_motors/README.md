# モーターを動かす

モーターは Raspberry Pi の GPIO から直接動かしません。必ずモータードライバと外部電源を使います。

## DC モーター

DC モーターは電圧をかけると回転します。PWM を使うと速度を変えられます。

必要なもの:

- DC モーター
- モータードライバ
- モーター用外部電源
- 共通 GND

## サーボモーター

サーボモーターは PWM 信号で角度を指定します。小型サーボでも電源は Raspberry Pi から直接取らず、外部 5V 電源を使う構成を推奨します。

## ステッピングモーター

ステッピングモーターは決まった角度ずつ回転します。位置制御に向いていますが、専用ドライバと適切な電源が必要です。

## 電源の注意

- モーター用電源と Raspberry Pi の GND を共通にする
- モーター電源のプラスを GPIO に接続しない
- モーターの定格電圧と定格電流を確認する
- 配線変更時は電源を切る
