# 全体像

この教材では、Raspberry Pi 5 を「測る」「判断する」「動かす」ための小さな実験台として使います。

## 学習の流れ

1. 電圧、電流、抵抗、GND、オームの法則を学ぶ
2. GPIO と PWM の役割を知る
3. ADC0834 を使ってアナログ電圧をデジタル値に変換する
4. センサー値を Python で読み取り、単位や意味に変換する
5. PWM とモータードライバでモーターを安全に動かす
6. センサーとモーターを組み合わせた制作課題に進む

## Raspberry Pi 5 でアナログを扱う考え方

Raspberry Pi 5 の GPIO はデジタル入力とデジタル出力です。Arduino のようなアナログ入力ピンはありません。

そのため、可変抵抗や光センサーのように連続的な電圧を出す部品を扱うには、ADC を使います。ADC は Analog-to-Digital Converter の略で、電圧を数値に変換する部品です。この教材では SunFounder Da Vinci Kit に含まれる ADC0834 を使います。

## この教材で重視すること

- 配線を見て意味が説明できること
- センサー値をただ読むだけでなく、電圧や物理量として考えること
- モーターを安全に動かすための電源設計を理解すること
- コードと回路を同時に少しずつ変えながら観察すること
