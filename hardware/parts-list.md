# 部品リスト

## 最小セット

| 種類 | 例 | 用途 |
| --- | --- | --- |
| Raspberry Pi | Raspberry Pi 5 | 制御用コンピュータ |
| ADC | ADS1115 | アナログ入力 |
| ブレッドボード | 400穴以上 | 試作 |
| ジャンパーワイヤ | オス-オス、オス-メス | 配線 |
| 抵抗 | 150 ohm、220 ohm、1k ohm、10k ohm | LED、分圧、プルアップ |
| LED | 任意色 | 出力の確認 |
| 可変抵抗 | 10k ohm | アナログ入力の練習 |

## センサー

| センサー | 学べること |
| --- | --- |
| CdS セル | 抵抗変化、分圧、明るさ |
| サーミスタ | 温度、非線形な変換 |
| 土壌水分センサー | キャリブレーション、環境変化 |
| 圧力センサー | アナログ値のしきい値判定 |
